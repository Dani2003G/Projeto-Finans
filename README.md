## 📁 Como baixar o projeto

```bash

# Clonar o repositório
$ git clone https://github.com/Dani2003G/Projeto-Finans

# Entrar no diretório
$ cd Projeto-finans

# Instalar as dependências
$ yarn install

# Iniciar o projeto
$ yarn start

```
